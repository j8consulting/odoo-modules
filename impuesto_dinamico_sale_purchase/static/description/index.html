Default Taxes in sale and purchase for Contacts

1. Enter to Contact Module

2. 2 New fields as added to the view "Sale Tax and Purchase Tax"

3. Select the tax to apply in sale and purchase order

4. Go to Sale or purchase Module and create a new order

5. Select a Contact

6. When selecting a contact, the previously selected tax appears in the sales/purchase tax field

7. Add a product to the order line and the tax is automatically assigned
